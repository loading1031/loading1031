# 🐾 커뮤니티 주인을 찾습니다, 묘집사

---

## 💡 기획 핵심

요즘 게임 중에 **커뮤니티 없는 게임은 없습니다.**  
방치형 게임조차 커뮤니티에서 이벤트를 진행합니다.  

저희는 여기서 **역발상**을 했습니다.

**게임 → 커뮤니티** ➡️ **커뮤니티 → 게임** ➡️ 커뮤니티 안에 게임을 넣자!  

---

### 🎮 게임 vs 🐰 묘집사 비교

| 게임 | 묘집사 |
|------|--------|
| 일반 / 메인 퀘스트 | 일반 / 메인 미션 |
| 보상 | 커뮤니티 관리자 권한 제공 (추후 보상 개발 예정) |
| STORY | [묘집사 STORY](https://www.notion.so/STORY-81fd8881a5bb45a7bfbc7b9a6ada61d5?pvs=21) |
| LOL 일반 게임 / 랭크 게임 | 일반 / 메인 미션 분리, 랭킹 도입 |

---

### 📌 커뮤니티(게임) 장점

1. **유저 주도 퀘스트 생성 및 완료**  
   - NPC 의존도를 줄이고, 유저 간 자율적인 플레이 구조 형성
2. **Business Model 확장**  
   - 코드 기반 아이템 판매 가능
3. **무한한 콘텐츠**  
   - 시즌제 도입 시, 매 시즌 신규 콘텐츠 반복 제공

---

### 🐇 묘집사의 의미

> **묘(卯)** = 토끼 묘  
> 앱의 마스코트이자 [STORY](https://www.notion.so/STORY-81fd8881a5bb45a7bfbc7b9a6ada61d5?pvs=21)의 중요한 요소

스토리를 따라 커뮤니티의 주인을 **묘집사**로 설정하였습니다.

2. 사용자와 **함께 성장하는 앱** 구축

## 🚀 다음 목표 (Next Goal)
